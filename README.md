# 3D Audio Visualizer

A browser-based music visualizer built with Three.js and Web Audio API.

## What it does
- Plays local audio files and reacts in real time
- Supports multiple visual layers (sphere, bars, particles, waveform, ring, tunnel, pulse plane)
- Includes bloom, themes, rainbow color cycling, and beat-reactive effects
- Offers camera choreography and section-aware auto switching
- Lets you save and load presets as JSON

## Controls
- **Space**: Play/Pause
- **F**: Fullscreen
- **M**: Mute
- **← / →**: Seek
- **↑ / ↓**: Volume

## GUI sections
- Visuals
- Look & Color
- Motion & Camera
- System

## Bar styles
- Radial
- Vertical
- Wall

## Quick start
- Clone the repo
- Open the project folder
- Run a local server (example):
  - `py -m http.server 5500`
- Visit:
  - `http://127.0.0.1:5500`

## Deploy
- Push to `main` to update GitHub Pages (if enabled)
